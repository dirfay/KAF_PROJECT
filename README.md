# KAF_PROJECT

Вебсайт кафедри систем штучного інтелекту та аналізу даних ТНТУ.

## Що є в проєкті

- головна сторінка;
- освітні програми;
- інформація для абітурієнтів;
- наукова діяльність;
- новини;
- контакти;
- стажування;
- розклад;
- адміністративна панель;
- інтелектуальний асистент;
- JSON-сховище знань.

## Технології

- Python
- HTML
- CSS
- JavaScript
- GitHub

## Призначення

Проєкт створений як сучасний кафедральний вебресурс із можливістю централізованого керування даними через адміністративну панель.

## Статус

Проєкт у розробці.
